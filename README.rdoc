== OL

This rails-api app parses the businesses.csv file and enters it into the Business model for use as an api.

Technologies used:

* ruby 2.3.1p112

* Rails 4.2.6

* will_paginate ~> 3.1.0

* RSpec

Instructions to run:

- pull repo
- `rake db:create`
- `rake db:migrate`
- `rails s`
- run app in browser on local port
